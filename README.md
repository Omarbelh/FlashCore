# FlashCore V1

FlashCore is a hybrid USB-C memory programmer combining the reliability of the CH341A with the flexibility of the ESP32-S3.

## Features

- USB-C interface
- CH341A programmer
- ESP32-S3 controller
- 320×240 TFT display
- Integrated microSD support
- Standalone operation
- SPI Flash support
- I²C EEPROM support (planned)

## Hardware

- CH341A
- ESP32-S3-WROOM-1
- AP2112K-3.3
- USBLC6-2SC6
- ILI9341 TFT
- USB-C
- Status LEDs
- Buzzer
- Push buttons
